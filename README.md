# 🌐 Navod Wijesooriya – Portfolio Website

A modern and responsive personal portfolio built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**.  
This portfolio highlights my background, featured projects, technical skills, and professional achievements in a visually appealing and interactive layout.

---

## ✨ Features

- ⚡ **Fast & Optimized** — Built with Vite for lightning-fast performance  
- 🎨 **Responsive Design** — Fully optimized for mobile, tablet, and desktop  
- 🌗 **Light/Dark Mode** — Toggle between light and dark themes  
- 💼 **Featured Projects Section** — Showcasing my academic and professional work  
- 🧠 **Core Strengths & Skills** — Clearly structured for recruiters and peers  
- 📬 **Contact Section** — Includes working email and social links  

---

## 🧰 Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS  
- **Build Tool:** Vite  
- **Version Control:** Git & GitHub
